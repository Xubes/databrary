DeferredAction

sealed class DeferredAction[R[_]] extends EssentialAction

An action that is created by pre-processing the RequestHeader alone first (with an empty body) using an ActionBuilder[R] to produce a R for input into the main action body. Basically this allows existing ActionBuilders to be used in cases when body processing should be deferred, and all the necessary content for the builder is in the header already.

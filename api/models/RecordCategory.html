RecordCategory

final class RecordCategory extends TableRowId[RecordCategory]

Types of Records that are relevant for data organization. Records that represent data buckets or other kinds of slot groupings (e.g., participants, days, conditions, etc.) can be assigned a particular RecordCategory for the purpose of display and templating. For now, all instances are hard-coded.

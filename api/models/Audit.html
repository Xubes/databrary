Audit

final case class Audit[T](when: Timestamp, who: Party.Id, ip: Inet, action: Audit.Action.Value, row: T) extends AuditBase with Product with Serializable

Represents an event row in an audit table of a particular type. Currently unused as there is intentionally no read/modify access to audit tables.

T

the type of data attached to this record. Usually this is a TableRow instance for a corresponding audit_t table.

when

the time of this event

who

the user who generated this event. May no longer be a valid party.

ip

the ip of the client machine that generated this event

action

the type of event

row

the remaining data columns
